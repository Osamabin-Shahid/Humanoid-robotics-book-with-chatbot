## Prerequisites
- Node.js 18+ installed
- Basic command line knowledge
- Git installed for version control
- Basic ROS 2 knowledge (Module 1 concepts)

## Setup Docusaurus Project (if not already done)

1. **Install Docusaurus globally**:
   ```bash
   npm install -g @docusaurus/init
   ```

2. **Navigate to your existing humanoid-robotics-ai-book directory**:
   ```bash
   cd humanoid-robotics-ai-book
   ```

3. **Install dependencies if needed**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm start
   ```

## Add Module 2 Content

1. **Create the module directory**:
   ```bash
   mkdir docs/module-2
   ```

2. **Add the four module files**:
   ```bash
   touch docs/module-2/intro.md
   touch docs/module-2/gazebo-physics.md
   touch docs/module-2/unity-visualization.md
   touch docs/module-2/sensor-simulation.md
   ```

3. **Create a category file for navigation**:
   ```bash
   # docs/module-2/_category_.json
   {
     "label": "Module 2: The Digital Twin (Gazebo & Unity)",
     "position": 3,
     "link": {
       "type": "generated-index",
       "description": "Introduction to physics-based simulation and digital twins using Gazebo and Unity"
     }
   }
   ```

## Build and Deploy

1. **Build the static site**:
   ```bash
   npm run build
   ```

2. **Deploy to GitHub Pages**:
   ```bash
   npm run deploy
   ```
